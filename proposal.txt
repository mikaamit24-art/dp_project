For my final project, I want to investigate whether customers of an online retail store can be grouped into meaningful behavioral segments based on their purchase history. My research question is: Can customers be segmented into distinct groups using transaction behavior, and how do these groups differ in value and repeat-purchase patterns?. I will use the Online Retail data set which contains many transactions. Since the data is recorded at a transaction level I will have to spend time to process the data to be at the consumer level. This will consist of cleaning data, removing cancelled transactions and feature engineering. The behavioural features I would need to create would be recency(how soon the customer made an order), frequency of purchase, monetary value of total orders, average order value and number of unique products purchased. I will use the k-means clustering approach to perform the segmentation. I will then look at each clusters perfomance on each of the behavioural features and form customer behavioural pattern labels.
